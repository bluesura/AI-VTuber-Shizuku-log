## SBVクリーニングプロンプト（2-1）改訂版

### このプロンプトの目的

`.sbv.txt`（YouTube自動生成字幕・SubViewer形式）を受け取り、**クリーン済み中間ファイル**を出力する。 次工程（2-2）で話者識別・第2層誤認識訂正を行うため、ここでは**テキスト内容を変えず、書式だけを整える**。

---

### ⛔ 絶対禁止事項

1. **発言テキストを1文でも省略してはならない**
2. **発言内容をAIが要約・言い換えしてはならない**
3. **「（※〇〇な場面）」などのAIによる注釈を発言行に混入させてはならない**
4. **SBVに存在しないタイムスタンプを補間・推測して使用してはならない**

---

### 処理手順（厳密に順番どおりに実行すること）

#### ステップ1：ブロック構造の保持と書式ノイズの除去

SBVの各ブロックを「タイムスタンプ＋テキスト」のペアとして**内部的に保持したまま**処理する。 タイムスタンプは出力フォーマット変換（Step5）まで**捨てずに各テキスト断片に紐付けておくこと。**

除去するのは以下のみ。**テキスト内容は一切除去しない。**

- 行頭が半角スペース1文字のみの空行
- 完全な空行（ブロック区切りの認識には使うが、最終出力には含めない）

タイムスタンプ行（例：`0:02:17.239,0:02:21.110`）は、**この段階では除去せずに対応テキストと紐付けて保持する。**

#### ステップ2：重複テキストの完全除去（タイムスタンプを引き継ぐ）

YouTubeの自動生成字幕では、スクロール表示のために**全く同じテキストが連続するブロックで何度も繰り返される**。 重複を除去するとき、**残すのは最初に出現したブロックのタイムスタンプとテキストのペア**とする。

具体例：

```
（ブロックA）0:00:05 → "俺が目さんこんにちは調子はいいですよ"  ← 残す（最初の出現）
（ブロックB）0:00:07 → "俺が目さんこんにちは調子はいいですよ"  ← 除去（完全重複）
（ブロックC）0:00:09 → "俺が目さんこんにちは調子はいいですよありがとう" ← 「ありがとう」が新規追加分
```

新規追加分の判定：前のブロックのテキストを**前方から完全一致で差し引いた残り**が新規テキストである。 新規テキストには、**そのブロック自身のタイムスタンプを紐付ける。**

#### ステップ3：発言断片の結合（タイムスタンプは先頭断片のものを継承）

同一発話の断片を**句点（。）または明確な文の区切り**を基準に1行に繋ぐ。

- 結合した行のタイムスタンプは、**結合した断片群の中で最初の断片のタイムスタンプ**を使う。
- 結合は形式的な連結のみ。**テキスト内容を変えてはならない。**
- `[音楽]` がテキスト途中に埋め込まれている場合（例：`ご主人様を[音楽]大切に`）は、**`[音楽]`を除去してテキストを繋ぎ、`ご主人様を大切に`** とする。ただし**その発言の文意が\[音楽\]なしで通るときのみ**この処理をする。文意が不明な場合は `[音楽]` を残す。
- 音楽のみのブロック（発言テキストがなく `[音楽]` しかないブロック）は **`🎵 [音楽]`** として独立行で残し、そのブロックのタイムスタンプを紐付ける。

#### ステップ4：第1層の誤認識訂正のみ適用

辞書に載っている語を**注記なしで**そのまま置換する。

| 字幕上の誤認識表記 | 正しい表記 | 備考 |
| --- | --- | --- |
| 秋先生 / 脇先生 / 秋先生 / 明先生 / あ先生 | あき先生 | 開発者名 |
| しず子 / しず / 雫 | しずく | 配信者名 |
| 同説 | 同接 | 同時接続数の略 |

> ⚠️ 第2層（チャット根拠による訂正）は**このプロンプトでは行わない**。次工程（2-2）で実施する。

#### ステップ5：タイムスタンプの出力形式への変換

ここまでの処理で各テキスト行に紐付けられたタイムスタンプを、出力形式 `[HH:MM:SS]` または `[MM:SS]` に変換して出力する。

**タイムスタンプ密度の方針**（ログ目的のため密に打つ）：

- 発話が途切れる区切り（無音・`🎵 [音楽]` 区間の後）ごとに打つ
- 同一発話が続く場合は**30〜60秒に1回**を目安に打つ（省略するときも、紐付けられたタイムスタンプからずらしてはならない）
- 話題の明確な切り替わりごとに必ず打つ

> ⚠️ **出力するタイムスタンプは必ずSBVに実在するもののみ使うこと。** タイムスタンプを間引く際は「その行に紐付けられたタイムスタンプ」を出力するか省略するかの二択であり、別の行のタイムスタンプで代用してはならない。

---

### 出力形式

```
[HH:MM:SS]
発言テキスト（クリーン済み・1文1行）
発言テキスト（続き）

[HH:MM:SS]
🎵 [音楽]

[HH:MM:SS]
発言テキスト
```

- AIによる注釈・補足は一切含めない

---

### 出力前の自己チェック

- [ ] すべての発言テキストが出力に含まれているか（抜け漏れゼロ）
- [ ] AIが発言内容を要約・言い換えていないか
- [ ] SBV特有の重複テキストが正しく除去され、新規テキストのみが繋がっているか
- [ ] 各行のタイムスタンプは、その行の**最初の断片が属していたSBVブロックのタイムスタンプ**と一致しているか
- [ ] SBVに存在しないタイムスタンプを補間・推測して使っていないか
- [ ] 第1層（辞書）の置換が正しく適用されているか
- [ ] 音楽区間が `🎵 [音楽]` として正しく残っているか
- [ ] タイムスタンプが適切な密度で打たれているか
- [ ] ポリシー制限コンテンツを削除・省略せず `（ガイドライン違反）` に置換しているか
- [ ] `（ガイドライン違反）` の置換が過剰になっていないか
